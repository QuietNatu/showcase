# Color

Several themes are available, each with it's own color palette. Use the theme dropdown in the menu bar to switch between them.

Each color has 11 shades with **50** being the main shade.
Colors must adhere to **WCAG 2.2 AAA** contrast requirements. For this, the main shade of each color is guaranteed to have a **7:1 contrast ratio** with the text color of it's theme.
